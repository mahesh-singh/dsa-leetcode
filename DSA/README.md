#### Complexity
- Search
  - Singly O(n)
  - Doubly O(n)
- Insert at head
  - Singly O(1)
  - Doubly O(1)
- Insert at tail
  - Singly O(1) 
  - Doubly O(1) 
- Remove at head
  - Singly O(1)
  - Doubly O(1)
- Remove at tail
  - Singly O(n) : because need to find new tail
  - Doubly O(1) : because tail have previous pointer which will become the new tail
- Remove at middle
  - Singly O(n) 
  - Doubly O(n)
